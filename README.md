To-Do List com HTML, CSS, JavaScript e SEO básico

Demo online: https://druuida.github.io/To_Do_List/

Projeto simples de To-Do List desenvolvido para praticar HTML, CSS, JavaScript, responsividade e conceitos básicos de SEO on-page.

# Funcionalidades
- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas da lista
- Modo claro/escuro (theme toggle)
- Layout responsivo

# Conceitos trabalhados

- HTML semântico (`main`, `header`, `aria-label`, etc.)
- SEO básico:
  - `title` descritivo
  - `meta description`
  - uso de headings (`h1`, texto descritivo)
- CSS:
  - variáveis de cor (`:root`)
  - modo claro/escuro usando a classe `body.dark`
  - media queries para mobile
- JavaScript:
  - manipulação do DOM
  - eventos (`submit`, `click`)
  - `localStorage` para salvar o tema escolhido
